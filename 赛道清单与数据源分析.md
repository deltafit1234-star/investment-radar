## 二、P0 赛道详解

### 2.1 AI大模型及应用层

#### 2.1.1 赛道简介

AI大模型是当前科技产业最核心变量，覆盖：基础模型（LLM、VLM）、开源模型、应用层（Agent、工具调用垂直应用）。

#### 2.1.2 关键细分方向

```
AI大模型
├── 基础模型（GPT-4、Claude、Llama、通义、文心等）
├── 开源社区（GitHub模型发布、Llama系、Stable Diffusion系）
├── 应用层（AI+办公、AI+代码、AI+搜索、AI+创作）
├── AI Agent（AutoGPT、Claude Agent、字节Coze等）
└── 端侧模型（移动端、嵌入式）
```

#### 2.1.3 数据源清单

| 数据源 | 类型 | 优先级 | 采集方式 | 免费程度 |
|--------|------|--------|---------|---------|
| GitHub Trending | 项目热度 | P0 | GitHub REST API | 免费（限速） |
| GitHub Stars | 单项目变化 | P0 | GitHub GraphQL API | 免费（需认证） |
| arXiv (cs.AI/cs.CL) | 论文 | P0 | arXiv API | 完全免费 |
| HuggingFace Models | 模型发布 | P0 | HF API | 免费 |
| OpenAI Blog | 动态 | P1 | RSS/爬虫 | 免费 |
| 智谱AI博客 | 国内动态 | P1 | RSS/爬虫 | 免费 |
| 机器之心 | 行业媒体 | P1 | RSS | 免费 |
| 量子位 | 行业媒体 | P1 | RSS | 免费 |

#### 2.1.4 关键监控指标

- **GitHub**：Star增长、PR活跃度、issue响应速度、contributor数量变化
- **论文**：按关键词过滤（新方法、新架构、新benchmark）
- **模型发布**：新模型上线、模型能力评测、开放程度变化
- **应用层**：爆款产品出现、用户增长数据（公开披露）

#### 2.1.5 信号示例

```
🚨 [AI大模型] Signal Detected
项目: llama3
事件: 7天Star增长 +45%（+12k stars）
活跃度: PR merge速度 +30%
可能含义: 开源社区正在快速跟进，可能对闭源模型形成压力
关联: 可能影响你们投的应用层公司估值逻辑

📊 [AI大模型] Data Point
本月新开源模型: 23个（环比+60%）
值得注意: 国产模型（Qwen、DeepSeek）占比提升至40%
```

### 2.2 半导体/算力

#### 2.2.1 赛道简介

半导体/算力是国产替代核心战场，涵盖：芯片设计（GPU/CPU/AI芯片）、晶圆制造封装、EDA工具、算力基础设施。

#### 2.2.2 关键细分方向

```
半导体/算力
├── AI芯片（GPU、ASIC、TPU）
├── 国产替代（CPU、存储、模拟芯片）
├── 先进封装（Chiplet、HBM）
├── EDA工具
├── 半导体设备
└── 算力租赁/云厂商
```

#### 2.2.3 数据源清单

| 数据源 | 类型 | 优先级 | 采集方式 | 免费程度 |
|--------|------|--------|---------|---------|
| 36kr 科技 | 融资新闻 | P0 | RSS/爬虫 | 免费 |
| IT桔子 | 融资数据库 | P0 | 网站爬虫 | 有限制 |
| 半导体行业观察 | 行业媒体 | P0 | RSS | 免费 |
| 芯东西 | 行业媒体 | P0 | RSS | 免费 |
| 美国出口管制网站 | 政策 | P0 | 官网监控 | 免费 |
| 科创板/证监会 | 上市动态 | P1 | 官网公告 | 免费 |
| 集微网 | 行业媒体 | P1 | RSS | 免费 |

#### 2.2.4 关键监控指标

- **融资**：融资金额、估值变化、投资方背景（是否头部VC）
- **政策**：美国BIS实体清单更新、出口管制新规、国内补贴政策
- **技术**：新芯片发布、代工厂产能变化、良率问题
- **上市**：科创板IPO受理、问询更新

#### 2.2.5 信号示例

```
🚨 [半导体] Alert
事件: XX公司获XX亿人民币B轮融资
投资方: 红杉、高瓴/头部VC
估值: 单方面XX亿（较上轮+XX%）
含义: 头部VC重仓，可能重新定价赛道

📜 [半导体] Policy Alert
美国BIS新增12家中国AI芯片公司至实体清单
影响评估: 国产替代逻辑进一步强化，利好非美系供应链

📊 [半导体] Data Point
本月国内AI芯片赛道融资: 8笔（环比+60%）
总金额: XX亿人民币
明星项目: XX（GPU）、XX（ASIC）
```

## 三、P1 赛道详解

### 3.1 机器人/具身智能

#### 3.1.1 赛道简介

2024-2025年大模型最直接的应用出口，人形机器人、具身智能硬件是新兴投资热点。

#### 3.1.2 关键细分方向

```
机器人/具身智能
├── 人形机器人（整机）
├── 灵巧手/执行器
├── 低速无人驾驶（配送、清扫）
├── 工业机器人
├── 核心零部件（减速器、传感器、控制器）
└── 具身智能算法
```

#### 3.1.3 数据源清单

| 数据源 | 类型 | 优先级 | 采集方式 | 免费程度 |
|--------|------|--------|---------|---------|
| GitHub 机器人相关 | 开源项目 | P0 | GitHub API | 免费 |
| arXiv (cs.RO/cs.AI) | 论文 | P0 | arXiv API | 免费 |
| ICRA/IROS顶会 | 学术动态 | P1 | 官网RSS | 免费 |
| 机器之心（机器人频道） | 行业媒体 | P1 | RSS | 免费 |
